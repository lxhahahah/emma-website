## 🚀 立刻开始（三选一）

### 选项 1: 直接打开（最快）
```bash
open /tmp/personal-website/index.html
```
⏱️ 30 秒

### 选项 2: 本地服务器（推荐）
```bash
cd /tmp/personal-website && python -m http.server 8000
```
然后打开 http://localhost:8000  
⏱️ 1 分钟

### 选项 3: 部署到网络
推送到 GitHub（username.github.io）  
⏱️ 5 分钟

## ⚠️ 重要事项

- 📍 所有数据保存在浏览器（localStorage）
- 🔄 不会自动备份，定期导出重要数据
- 🗑️ 删除操作无法撤销
- 📱 完全响应式，可用手机访问
