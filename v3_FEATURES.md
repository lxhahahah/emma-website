```
1. 进入 "写文章" 页面
   ↓
2. 填写表单（标题、内容、标签、分组等）
   ↓
3. 点击 "发布文章"
   ↓
```

### 3. 关键词云怎么工作？

- 自动分析所有文章的文本内容
- 计算每个词出现的频率
- 显示最常见的 15 个关键词
- 词的大小 = 出现频率（频率越高词越大）
- 会自动排除常见词（的、了、是等）

### 4. 标签和分组的区别

| | 标签 | 分组 |
|---|---|---|
| **数量** | 多个 | 一个 |
| **用途** | 细粒度分类 | 大分类 |
| **示例** | 学习、技术、生活 | 📚 学习 |
| **作用** | 搜索和过滤 | 组织和分类 |

## 💾 数据存储

### 本地存储（默认）
- 文章保存在浏览器 localStorage
- 仅在当前浏览器有效

- 不需要任何配置

### Firebase（可选）
- 启用后数据存储到云端
- 所有设备可以同步访问
- 配置方法见 `FIREBASE_SETUP.md`

## 🐛 常见问题

### Q1: 关键词云显示不出来？
**A:** 只有发布了足够多的文章才会生成关键词云。多发布几篇文章试试！

### Q2: 编辑后能恢复吗？
**A:** 不能。编辑是新建文章，老文章会被删除。建议重要文章修改前备份。

### Q3: 标签和分组哪个我应该用？
**A:** 两个都用最好！分组用来大分类（学习/生活），标签用来细分类（Python/CSS）。

### Q4: 如何导出所有文章？
**A:** 目前没有自动导出功能。可以从浏览器开发工具导出 localStorage：
```javascript
// F12 → 控制台 → 粘贴：
copy(JSON.parse(localStorage.getItem('blogPosts')))
```

### Q5: 删除了文章能恢复吗？
**A:** 本地存储无法恢复。使用 Firebase 可能有备份。下次记得备份重要文章！

## 📚 技术细节

### 新增函数

- `showArticleDetail()` - 显示文章详情
- `editCurrentArticle()` - 编辑当前文章
- `deleteCurrentArticle()` - 删除当前文章
- `generateWordCloud()` - 生成关键词云
- `analyzeKeywords()` - 分析关键词频率
- `generateTags()` - 生成标签过滤按钮
- `filterByTag()` - 按标签过滤
- `renderBlogPosts()` - 渲染文章列表（支持过滤）

### 数据结构

```javascript
{
    id: 1724094245000,           // 时间戳 ID
    title: "文章标题",             // 必填
    date: "2024-04-17",           // 发布日期
    emoji: "📱",                  // 文章表情
    excerpt: "摘要...",            // 简短描述
    content: "完整内容...",         // 文章正文
    tags: ["标签1", "标签2"],      // 标签数组
    category: "tech"              // 分组类别
}
```

### 浏览器路由

- `#home` - 首页
- `#about` - 关于页面
- `#blog` - 博客列表
- `#write` - 写文章
- `#post-123` - 文章详情（123 是文章 ID）

## 🎨 自定义指南

### 修改分组类别

编辑 `script.js` 中的 `categoryMap`：

```javascript
const categoryMap = {
    'learning': '📚 学习',
    'life': '🌟 生活',
    // 添加新分组...
    'custom': '🎯 自定义'
};
```

同时在 HTML 中的 `postCategory` 选择框添加选项。

### 修改关键词过滤

编辑 `script.js` 中的 `stopWords` 集合，添加或删除要排除的词。
