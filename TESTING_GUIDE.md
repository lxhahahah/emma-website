## 完整操作流程

### 流程 1：发布 → 查看 → 编辑 → 删除

```
1. 进入 "写文章" 页面
2. 填写：
   - 标题：我的测试文章
   - 表情：🎯
   - 内容：这是一篇测试文章，用来检验所有功能
   - 标签：测试, 功能检验
   - 分组：💭 思考
3. 点击 "发布文章"
4. 等待提示，自动进入博客页面
5. 找到新发布的文章卡片
6. 点击卡片进入详情页
7. 验证标题、内容、标签都正确显示
8. 点击 "编辑" 按钮
9. 修改标题为 "我的测试文章 - 已编辑"
10. 点击 "发布文章"
11. 回到博客页面验证更新
12. 再次进入文章详情页
13. 点击 "删除" 按钮
14. 确认删除
15. 回到博客页面，文章消失
```

### 流程 2：标签过滤

```
1. 进入 "博客" 页面
2. 向下找到标签过滤按钮
3. 如果有多个标签，点击第一个标签
4. 文章列表只显示包含该标签的文章
5. 点击该标签再次取消过滤
6. 所有文章重新显示
```

### 流程 3：关键词分析

```
1. 发布至少 3 篇文章，包含不同内容
2. 进入 "博客" 页面
3. 向下找到 "📊 关键词云"
4. 查看显示的关键词
5. 验证关键词大小与出现频率对应
6. 鼠标悬停查看频率提示
```

## 故障排查

### 关键词云不显示？
1. 检查是否发布了文章
2. 确认文章内容长度足够（>50 字）
3. 刷新页面重试
4. 查看浏览器控制台是否有错误

### 编辑按钮不工作？
1. 确认在文章详情页
2. 检查浏览器控制台错误
3. 尝试刷新页面
4. 确认 localStorage 有空间

### 标签过滤不工作？
1. 确认文章有标签
2. 检查标签写法（不要有多余空格）
3. 刷新页面重试
4. 查看标签按钮是否显示

### 数据丢失？
1. 检查浏览器隐私模式
2. 检查 localStorage 是否被清除
3. 打开浏览器开发工具查看存储
4. 考虑启用 Firebase 备份
